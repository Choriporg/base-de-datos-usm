Este proyecto crea una base de datos sobre los mundiales de 1930 en adelante, obteniendo la información de archivos .csv

Este programa requiere tener instaladas la bibiolteca pandas y pyodbc. Además es necesario tener instalado SQL y
SQL SERVER MANAGMENT STUDIO

En caso de que estas librerias no se encuentren instaladas con los siguientes comandos:

pip install pandas
pip install pyodbc

La ejecución del programa puede hacerse por medio del interprete de python o por consola con el comando:

python3 <<Nombre del programa>>.py
